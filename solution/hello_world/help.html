ThirdLight File Explorer

Help

This app it's very easy to work with.

The app is divided in three parts:

  1. Breadcrumb: That tells you where are you and you can navigate by clicking where you want to go.
  2. Search input: Where you can filters the results on the directory you are viewing.
  3. Content: Where you see the content of the directory or the file you selected.

Breadcrumb

The breadcrumb will tell you the directory you are seen on the content area.

You can move between directories (going back in the tree) using the breadcrumb by clicking on the name of the folder that you want to go.

Search input

Maybe, a folder that you are seen will have too much content on it.

In order to help you find an element inside the folder you can type the name or letters contained on the item that you want to search.

The search input will filter results on real time.

Content area

The content area will show you files and folders that are inside the folder you are viewing in a table.

Different and well know icons will identify if it's a folder or a file.

You can see the content area in two ways. See the folders or see the detail of the file and the folder too.

It means, when you click on a file, the details will apear at right with a button where you can download (this doesn't work because the JSON file doesn't give you a link, but you can make an idea about how it can be).

If you click on a directory/folder you will go inside them.

UX/UI

The best way to know the UX on this app it's to see the "paper image prototypes". Do you want to see the prototype of the application?.

On the UI I have choosen as a base Bootstrap HTML framework to help and be more fast on the layout of the application but I have change the colors to a green and black and white style.

Black and white are the most easy election because I don't know the color palete that will be the best. Because, should I choose the business colors or make by my own?.

So my first idea it's to use the color that match the Logo of ThirdLight (as can see on the header) and then a black and white palete, but with that election can be "old", so to have a point of color I choose a dark green because green it's the color of nature, energy and safety. Using a folder application must be "safety" for the user.

Icons in black and white and a white background for the content will focus the user on the data, so that's why I choose not to put colors except a brief green when you see the item content, to help the user be more confortable.
